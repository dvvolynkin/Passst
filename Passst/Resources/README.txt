Passst resource directory.

PassstIcon.svg is the color app-icon master. PassstMarkMono.svg uses the same
passing-card mark as a flat template for the menu bar. Run
scripts/generate-icons.sh after changing either master; it regenerates the
asset-catalog PNGs, Passst.icns, and PassstMenuBarTemplate.png.

ClipboardCopy.wav is the original tactile clipboard-capture effect. Run
scripts/generate-sounds.swift to regenerate it deterministically.
